
  # GYMUP

  Prototipo web de GYMUP para tracking de entrenamientos de gimnasio, exportado desde Stitch/Figma y saneado para desarrollo local con React + Vite.

  ## Scripts

  - `npm install`: instala dependencias
  - `npm run dev`: levanta el entorno de desarrollo y abre el proyecto en el navegador
  - `npm run build`: genera el build de producción
  - `npm run typecheck`: corre chequeo de tipos con TypeScript
